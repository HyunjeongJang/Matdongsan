## 환경

- **Back End** : Java11, Spring, Spring Security
- **Front End** :  JavaScript, Html/CSS
- **DevOps** : ORACLE 11g , Oracle Cloud , Redis, AWS Ec2, Route53
- **Collaboration** : Git , Notion

## 서비스 소개

### matdongsan.site
부동산 가격 정보를 확인하고, 지역 커뮤니티를 통해 동네 주변 정보를 알아볼 수 있는 사이트 입니다.
